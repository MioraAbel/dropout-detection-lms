# Dropout Detection LMS

Système prédictif basé sur le Machine Learning pour la détection précoce du décrochage dans les plateformes LMS.

## Stack technique
- Backend : Python, Flask, scikit-learn, XGBoost
- Base de données : MySQL
- Notebook : Jupyter

## Structure du projet
- backend_lms/ : API REST Flask
- frontend_lms/ : Interface utilisateur
- notebooks/ : Pipeline ML
- models/ : Modèle entraîné (le choix final)
- data/ : Dataset (confidentiel)
